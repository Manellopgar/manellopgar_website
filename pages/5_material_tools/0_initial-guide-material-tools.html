Initial Guide for All Materials

Here is an explanation of how to add any Material Tool.

? How to add any Material Tool?
▼

Select any mesh object, go to the Materials Addon panel and click any Material Tool button. The selected Material Tool is added automatically.

Modifier Tools Info

The Global Material, Multishader, Mapping ColorToInstances, and Override Multishader tools are Modifier tools. The other tools do not have a Geometry Node modifier.
