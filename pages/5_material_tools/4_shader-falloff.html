Shader Falloff

Adds a pre-made material to the selected object, featuring a template configuration with two materials using the Gradient Falloff from B4D Tools.

The material is named SHDR_Falloff_Template. It contains two Principled BSDF nodes connected via a Mix Shader. The Factor input uses an Attribute node set to Instancer Mode. Below this, you will find the same setup configured for Geometry Mode, which is used if you utilize the Realize Instances tool or modifier.
