Shader Falloff

Adds to the selected Object a pre-made material template that configures two materials using the Gradient Falloff of the B4D Tools.

The material is called SHDR_Falloff_Template. In this setup, you will see two Principled BSDF nodes connected via a Mix Shader. The Factor input utilizes an Attribute in Instancer Mode, or the same configuration in Geometry Mode if you use the Realize Instances Tool/Modifier.

Configured Material Setup

You can also use any other named Attribute from B4D Tools. You can always check the available attributes in the latest Modifier of B4D in the Modifier's stack.
