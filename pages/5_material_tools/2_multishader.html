Set Multishader MatSlots

Note: This section requires insertion from 0a_Material_Tools_Params.txt.

Only Material Slots

Configure the Material Slots.

  • Disable: The Multishader assigns each Material Slot in the current order to each Instance.
  • Enable: The Multishader assigns each Material Slot in the current order to each Instance. But you can control how is assigned, changing the Amount, the First Material and the Offset.

First Mat Slot

Set the First Material Slot Index that will be the First Material used.

Nº of Mat Slots

Set the Total Amount of Material Slots that you have in the current Object.

Offset Mat Slots

Adds Offset to the current Material Slot Index.

Randomize Materials

Randomize the Material Slot Indexes.

  • Disable: The Randomize is disabled.
  • Enable: The Randomize is enabled.

Sort Randomize

ReRandomize the half of the Instances. Helpful to avoid repetition.

  • Disable: The Randomize is disabled.
  • Enable: The Randomize is enabled.

Seed

Set Seed of the Randomize Materials.

Keep Mesh (Not Instances)

Keep the No Instance Geometry.

Split Selection

Use Custom Selection for Split to Instances. You can use Input Attribute Toggle.

Use Island Index

Use Island Index for Split to Instances.
