Set Multishader MatSlots

This tool allows you to assign each material slot to each instance in index order. By default, it assigns all material slots.

Material Slot Configuration

  • Only Material Slots: Configure the Material Slots.
    • Disable: The Multishader assigns each Material Slot in the current order to each Instance.
    • Enable: The Multishader assigns each Material Slot in the current order to each Instance. However, you can control the assignment by changing the Amount, First Material, and Offset.
  • First Mat Slot: Set the First Material Slot Index that will be the first material used.
  • Nº of Mat Slots: Set the Total Amount of Material Slots available in the current Object.
  • Offset Mat Slots: Adds an Offset to the current Material Slot Index.

Material Slots - Parameters

Randomization

  • Randomize Materials: Randomize the Material Slot Indexes.
    • Disable: The Randomize is disabled.
    • Enable: The Randomize is enabled.
  • Sort Randomize: Re-randomize half of the Instances. Helpful to avoid repetition.
    • Disable: The Randomize is disabled.
    • Enable: The Randomize is enabled.
  • Seed: Set the Seed of the Randomize Materials.

Material Slots - Random Parameters

Mesh & Split Options

  • Keep Mesh (Not Instances): Keep the Non-Instance Geometry.
  • Split Selection: Use Custom Selection for Split to Instances. You can use Input Attribute Toggle.
  • Use Island Index: Use Island Index for Split to Instances.

Multishader with Multiple Assets from Collection

This is an example of how to use the Multishader when you need to use Assets from a Collection.

  • 1st: Add the Multishader Modifier.
  • 2nd: You must have selected the first asset of the Collection in the Outliner.
  • 3rd: Add as many materials as you need, keeping in mind that each slot has its own material.

Assign More Than 1 Multishader

This is an example of how to use more than 1 Multishader for the same BCloner/BCloneCutter.

  • 1st: Add one selection for Front and another for Back in the Custom Selection. Enable Only Material Slots if needed.
  • 2nd: You must enable the Split to Instances Advanced parameter and the Split Selection. If it doesn't work, also enable Use Island Index. This is mandatory for all Multishader modifiers except the last one.
  • 3rd: Verify everything is configured correctly. Don't forget to check if you need to fix the direction of the Back textures.
