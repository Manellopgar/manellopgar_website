Set Mapping Color To Instances

Note: This section requires insertion from 0a_Material_Tools_Params.txt.

Image

Set your Custom Image to Mapping the Color.

Color To

Choose the Color Mode.

  • Vertex: Mapping the Image Color to each Vertex of the Geometry.
  • Instances: Mapping the Image Color to each Instance.

Custom Material

Set your Custom Material.

Image Parameters

Image Advanced Parameters.

  • Disable: Disable the Advanced Parameters of the Image Input.
  • Enable: Enable the Advanced Parameters of the Image Input.

Frame

If the Image supports animation, the frame can be set here. This can be keyframed so that the image changes between frames.

Offset

Changes the Position of the Image.

Rotation

Changes the Rotation of the Image.

Scale

Changes the Scale of the Image.

Uniform Scale

Changes the Uniform Scale of the Image.
