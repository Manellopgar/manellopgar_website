Set MappingColor To Instances

This tool allows mapping the color of images to instances.

Image: Set your Custom Image to map the color.

Color To: Choose the Color Mode

  • Vertex: Maps the Image Color to each vertex of the geometry.
  • Instances: Maps the Image Color to each instance.

Mapping Color To Instance - Example

Custom Material: Set your Custom Material.

Image Parameters: Advanced parameters for the Image Input.

  • Disable: Disables the Advanced Parameters of the Image Input.
  • Enable: Enables the Advanced Parameters of the Image Input.

Frame: If the image supports animation, the frame can be set here. This can be keyframed so that the image changes between frames.

Offset: Changes the Position of the Image.

Rotation: Changes the Rotation of the Image.

Scale: Changes the Scale of the Image.

Uniform Scale: Changes the Uniform Scale of the Image.
