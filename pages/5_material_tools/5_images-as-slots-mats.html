Images as SlotMats

This tool is highly effective for adding multiple images to any BCloner. Upon clicking, an import window appears. It offers functionality similar to Blender’s native Mesh Plane tool, but instead of generating a mesh plane for each image, it creates one material slot per image and adds them to the selected object or B4D Tool.

Import Settings

When you click the Images as MatSlots button, the importer window opens. Navigate to your custom folder containing all the images.

Video Settings

Here you will find several parameters to configure before importing video textures. These are compatible with video file types supported by Blender.

Start Frame: Defines the starting frame for the video textures.

Cyclic: Enables the Cyclic option in the Image Texture Node for each video texture.

Auto Refresh: Enables the Auto Refresh option in the Image Texture Node for each video texture.

Randomize Frame Range: Randomizes the frame range of each video texture to avoid repetition. This is useful if you have the same animation for all video textures.

Use Time Remapping: This is not a native feature of the Image Texture Node; it is a workaround that allows each video texture to be animated with default time and duration. It is useful if you want to change the speed or duration for all videos later without re-importing all video textures.

Image Settings

Here you will find parameters to configure before importing images. Please note that these settings are currently unavailable and will be added in a future update.

Shader Configuration

All created materials follow the same structure:

1. A node group called "SHDR_Custom_UVMapping" for UV Mapping, used in all materials. This allows you to make changes to all materials at once by editing this node group.

2. The Image Texture node with the corresponding image.

3. A node group called "SHDR_Custom_Shader" for the main Shader, used in all materials. This allows you to make changes to all materials at once by editing this node group.
