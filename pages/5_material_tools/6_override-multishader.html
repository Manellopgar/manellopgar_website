Override Multishader

Note: This section requires insertion from 0a_Material_Tools_Params.txt.

Material Selection

Set a Custom Selection for the Override Multishader Material. You can use Input Attribute Toggle.

Original BCloner

Set the Object with the Original BCloner/BFracture that has the Multishader added.

  • Original: Output the instances relative to the collection offset.
  • Relative: Join the input collection instances with the modified object as geometry, maintaining the relative position between the objects in the scene.

Offset MatSlots

Adds Offset to the current Material Slot Index.

Transformation

Choose the Transformation Mode.

  • Original: Output the instances relative to the collection offset.
  • Relative: Join the input collection instances with the modified object as geometry, maintaining the relative position between the objects in the scene.

As Instance

Output the entire object as a single instance. This allows instancing non-geometry object types.
