Override Multishader

This tool simplifies the process of overriding an existing Multishader. It is particularly useful when you have different textures for different regions, but the clone setup remains the same. This allows you to easily switch between texture sets for different regions using a single tool.

Parameters

Material Selection: Set a custom selection for the Override Multishader Material. You can use Input Attribute Toggle if needed.

Original BCloner: Set the object with the original BCloner/BFracture that has the Multishader added.

  • Original: Output the instances relative to the collection offset.
  • Relative: Join the input collection instances with the modified object as geometry, maintaining the relative position between the objects in the scene.

Offset MatSlots: Adds an offset to the current Material Slot Index.

Transformation: Choose the Transformation Mode.

  • Original: Output the instances relative to the collection offset.
  • Relative: Join the input collection instances with the modified object as geometry, maintaining the relative position between the objects in the scene.

As Instance: Output the entire object as a single instance. This allows instancing non-geometry object types.
