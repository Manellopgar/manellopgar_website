Object BMograph Animator Tool

The Object BMograph Animator allows you to create dynamic animations for instances with advanced control over selection, mixing, and interpolation.

Selection & Mixing Controls

  • Custom Selection: Set a Custom Selection for the Instances. You can use Input Attribute Toggle.
  • Mix: Amount of mixing between the Static and Animation.

Min/Max Controls

  • Disable: Uses full values for the Animation.
  • Enable: Enables to adjust Min/Max values for the Animation.
  • Min: Adjust the Minimum Value for the Animation.
  • Max: Adjust the Maximum Value for the Animation.

Animator Type

Choose the Animator Type:

  • Vibrate: Can be used to add random behavior to the animation. Or with Regular Pulse a Cyclic Animation.
  • Repeat: Repeats the Animation configured on the Amplitude parameters.
  • Ping-Pong: Repeats Foward and Backwards the Animation configured on the Amplitude parameters.

Interpolation

Choose the Interpolation Type:

  • Linear: No Ease Interpolation.
  • Sine: Adds the Sine Easing Interpolation to the Animation.
  • Cubic: Adds the Cubic Easing Interpolation to the Animation.
  • Exponential: Adds the Exponential Easing Interpolation to the Animation.
  • Circular: Adds the Circular Easing Interpolation to the Animation.
  • Back: Adds the Back Easing Interpolation to the Animation.
  • Bounce: Adds the Bounce Easing Interpolation to the Animation.
  • Elastic: Adds the Elastic Easing Interpolation to the Animation.
  • Stepped Linear: Adds the Stepped Linear Easing Interpolation to the Animation. (Use Steps parameter to set number of steps).
  • Ease Quad: Adds the Ease Quad Easing Interpolation to the Animation.

Animation Settings

  • Steps: Set the Number of Steps for the Stepped Linear Interpolation.
  • Frequency (Speed): Control Speed Animation. INFO: Loop ONLY works perfectly with Integer values.
  • Frame Loop Out: To enable Loop Animation set the End Frame, that must be greater than 0.
  • Regular Pulse: If this option is enabled, the object pulsates using a regular pattern shaped like a sine wave.
  • Offset: Offset of Sine Animation.
  • Seed: Adjust the Seed from the Random Animation.

Transform Parameters

  • Use Position: Enables the Position Parameters.
  • Amplitude (Position): The Amplitude input boxes control the maximum positive and negative movement along the X, Y and Z axes. The Frequency value controls the speed of vibration.
  • Use Rotation: Enables the Rotation Parameters.
  • Amplitude (Rotation): The Amplitude input boxes control the maximum positive and negative rotation along the X, Y and Z axes. The Frequency value controls the speed of vibration.
  • Use Scale: Enables the Scale Parameters.
  • Amplitude (Scale): The Amplitude input boxes control the maximum positive and negative scale along the X, Y and Z axes. The Frequency value controls the speed of vibration.
  • Uniform Scale: The Uniform Scale option determines whether all axes are scaled uniformly.
  • Local Space: If enabled, the instances are scaled in local space. In other words, they are scaled in the directions described by the initial transform of each instance. When disabled, the Center and Scale inputs are specified in the local space of the modifier object.

Advanced Settings

Enable Advanced Settings:

  • Disable: Advanced Settings Disabled.
  • Enable: In Vibrate Mode enables the Easing Interpolation Controls. In the Other Modes, enables Prune Controls.

Once enabled (Vibrate Mode), you can control Easing:

  • Noise Random Scale: Overall Noise Texture scale.
  • Ease Mix: Adjust the amount of Ease.
  • Ease In-Out: Adjust the Percentage Between Ease In-Out.
  • To Min: Adjust the Minimum of the Ease for the Animation.
  • To Max: Adjust the Maximum of the Ease for the Animation.

Once enabled (Other Modes), you can control Pruning:

  • Prune Pos Amount: Controls the amount of Instances Prune for the Position Animation.
  • Prune Rot Amount: Controls the amount of Instances Prune for the Rotation Animation.
  • Prune Scale Amount: Controls the amount of Instances Prune for the Scale Animation.
  • Seed: Adjust the Seed from the Random Prune.
