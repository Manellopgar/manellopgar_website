Object BMograph Animator Tool

This tool allows you to animate any B4D Tools without using keyframes. It is a powerful utility that combines different modes and functions similarly to Cinema 4D's Vibrate Tag, but with significantly more features. It works with single object meshes as well as any BCloner or BFracture tool.

Core Parameters

Custom Selection: Set a Custom Selection for the Instances. You can use an Input Attribute Toggle.

Mix: Amount of mixing between the Static and Animation states.

Min/Max Settings

Min/Max Mode: Use the Minimum/Maximum amount for the Animation.

  • Disable: Uses full values for the Animation.
  • Enable: Enables the ability to adjust Min/Max values for the Animation.

Min: Adjust the Minimum Value for the Animation.

Max: Adjust the Maximum Value for the Animation.

Animator Type

Animator Type: Choose the Animator Type.

  • Vibrate: Can be used to add random behavior to the animation or create a cyclic animation with Regular Pulse.
  • Repeat: Repeats the Animation configured in the Amplitude parameters.
  • Ping-Pong: Repeats the Animation Forward and Backward based on the Amplitude parameters.
  • On/Off: Enables the configured on the Amplitude parameters until the Frame indicated.

Interpolation

Interpolation: Choose the Interpolation Type.

  • Linear: No Ease Interpolation.
  • Sine: Adds Sine Easing Interpolation to the Animation.
  • Cubic: Adds Cubic Easing Interpolation to the Animation.
  • Exponential: Adds Exponential Easing Interpolation to the Animation.
  • Circular: Adds Circular Easing Interpolation to the Animation.
  • Back: Adds Back Easing Interpolation to the Animation.
  • Bounce: Adds Bounce Easing Interpolation to the Animation.
  • Elastic: Adds Elastic Easing Interpolation to the Animation.
  • Stepped Linear: Adds Stepped Linear Easing Interpolation to the Animation.
  • Ease Quad: Adds Ease Quad Easing Interpolation to the Animation.

Animation Controls

Steps: Set the Number of Steps for the Stepped Linear Interpolation.

Frequency (Speed): Control Animation Speed. Note: Loop functionality works perfectly with Integer values.

Frame Loop Out: To enable Loop Animation, set the End Frame to a value greater than 0.

On/Off Frame (Disable): Set which frame will disable the Amplitude parameters.

Off/On Frame (Enable): Set which frame will enable the Amplitude parameters.

Regular Pulse: If enabled, the object pulsates using a regular pattern shaped like a sine wave.

Offset: Offset of the Sine Animation.

Seed: Adjust the Seed for the Random Animation.

Axis Parameters

Use Position: Enables the Position Parameters.

Amplitude: The Amplitude input boxes control the maximum positive and negative movement along the X, Y, and Z axes. The Frequency value controls the speed of vibration.

Use Rotation: Enables the Rotation Parameters.

Amplitude: The Amplitude input boxes control the maximum positive and negative rotation along the X, Y, and Z axes. The Frequency value controls the speed of vibration.

Use Scale: Enables the Scale Parameters.

Amplitude: The Amplitude input boxes control the maximum positive and negative scale along the X, Y, and Z axes. The Frequency value controls the speed of vibration.

Uniform Scale: Determines whether all axes are scaled uniformly.

Local Space: If enabled, instances are scaled in local space (described by the initial transform of each instance). When disabled, the Center and Scale inputs are specified in the local space of the modifier object.

Advanced Settings

Advanced Settings: Enable Advanced Settings.

  • Disable: Advanced Settings Disabled.
  • Enable: In Vibrate Mode, enables Easing Interpolation Controls. In Other Modes, enables Prune Controls.

Noise Random Scale: Overall Noise Texture scale.

Ease Mix: Adjust the amount of Ease.

Ease In-Out: Adjust the Percentage Between Ease In-Out.

To Min: Adjust the Minimum of the Ease for the Animation.

To Max: Adjust the Maximum of the Ease for the Animation.

Prune Pos Amount: Controls the amount of Instances Prune for the Position Animation.

Prune Rot Amount: Controls the amount of Instances Prune for the Rotation Animation.

Prune Scale Amount: Controls the amount of Instances Prune for the Scale Animation.

Seed: Adjust the Seed for the Random Prune.
