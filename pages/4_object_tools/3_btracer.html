Asset Type

Choose Asset Type:

  • Object: Uses the Trace Link Object.
  • Collection: Uses the Trace Collection.

Trace Link

Drag the Object whose points (polygonal object) are to be taken into account by the Tracer.

Trace Collection

Drag the Collection whose Objects are to be taken into account by the Tracer.

Tracing Mode

Choose the Tracing Mode:

  • Trace Paths: The vertex path of objects will be converted to splines.
  • Connect All Objects: The objects will be connected with a segment of a common spline.
  • Connect Elements: Objects from different elements drawn by the same tracer will be connected with a common spline but with different segments.
  • Connect All Points: All Points will be connected by same traer with a common spline.

Trace Active

If you want to switch off the Tracer temporarily, you can do so with this option. This creates gaps in the splines and thus spline segments.

Trace

Use this option to decide whether all objects should be recorded:

  • Origin: From the Object Center Point.
  • Vertices: From the Object Vertices.

Use Limit Age From End

Enables the Limit Age from End.

Limit Age From End

Splines are generated from the end of the animation for as long as you set frames with this parameter.

Use Limit Age From Start

Enables the Limit Age from Start.

Limit Age From Start

Splines are generated from the start of the animation for as long as you set frames with this parameter.

Spline Settings

  • Disable: (Value: 0)
  • Enable: (Value: 0)

Spline Type

  • Poly: This simplest of all spline types connects the support points that define the polygon using direct connecting lines. You can use these splines to create angular objects or to simulate choppy movements during animation.
  • Bezier: This type of spline creates a smooth curve between the support points, which can be controlled very precisely. The interpolated curve always passes exactly through the interpolation points.
  • NURBS: This type of spline creates a smooth curve between the points, spline gets auto handles.
  • Catmull Rom: This type of spline creates a smooth curve between the points, spline gets auto handles.

Close Spline

Each spline can be closed or open. If a spline is closed, the start and end points are connected.

Reverse Spline

  • None: No Spline Resampling.
  • Count: Sample the specified number of points along each spline.
  • Evaluated: Calculate the number of samples by splitting each spline into segments with the specified length. The length will be rounded down so that a whole number of samples will fit in each input spline.
  • Length: Evaluate the spline's points based on the resolution attribute for NURBS and Bézier splines. Changes nothing for poly splines.

Resample Mode

  • None: No Spline Resampling.
  • Count: Sample the specified number of points along each spline.
  • Evaluated: Calculate the number of samples by splitting each spline into segments with the specified length. The length will be rounded down so that a whole number of samples will fit in each input spline.
  • Length: Evaluate the spline's points based on the resolution attribute for NURBS and Bézier splines. Changes nothing for poly splines.

Resample Count

The number of control points on the new splines.

Resample Length

The approximate length between the control points of the new splines.
