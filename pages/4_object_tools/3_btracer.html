Object BTracer

This tool generates curves from animated objects, capturing data from vertices, origins, and other key points. It is designed to simplify the creation of tracer trails and motion paths.

BTracer - Animation Example

Asset Type

Asset Type: Choose the source type for the tracer.

  • Object: Uses the Trace Link Object.
  • Collection: Uses the Trace Collection.

Trace Source

Trace Link: Drag the object whose points (polygonal object) are to be taken into account by the Tracer.

Trace Collection: Drag the collection whose objects are to be taken into account by the Tracer.

Tracing Mode

Tracing Mode: Choose how the tracer processes the input data.

  • Trace Paths: The vertex path of objects will be converted to splines.
  • Connect All Objects: The objects will be connected with a segment of a common spline.
  • Connect Elements: Objects from different elements drawn by the same tracer will be connected with a common spline but with different segments.
  • Connect All Points: All points will be connected by the same tracer with a common spline.

BTracer - Trace Mode

Trace Options

Trace Active: If you want to switch off the Tracer temporarily, you can do so with this option. This creates gaps in the splines and thus spline segments.

Trace: Use this option to decide whether all objects should be recorded.

  • Origin: From the Object Center Point.
  • Vertices: From the Object Vertices.

BTracer - Trace Options

Time Limits

Use Limit Age From End: Enables the Limit Age from End.

Limit Age From End: Splines are generated from the end of the animation for as long as you set frames with this parameter.

Use Limit Age From Start: Enables the Limit Age from Start.

Limit Age From Start: Splines are generated from the start of the animation for as long as you set frames with this parameter.

BTracer - From Start/End

Spline Settings

Spline Settings: Controls the generation of spline data.

  • Disable: Disable the Spline Settings.
  • Enable: Enable the Spline Settings.

Spline Type: Choose the type of curve to generate.

  • Poly: The simplest spline type connects support points using direct lines. Useful for creating angular objects or simulating choppy movements.
  • Bezier: Creates a smooth curve between support points, controllable with high precision. The interpolated curve always passes exactly through the interpolation points.
  • NURBS: Creates a smooth curve between points with auto-generated handles.
  • Catmull Rom: Creates a smooth curve between points with auto-generated handles.

Close Spline: Determines if each spline is closed or open. If closed, the start and end points are connected.

Reverse Spline: Activates point order reversal for the internal spline.

  • None: No spline resampling.
  • Count: Sample the specified number of points along each spline.
  • Evaluated: Calculate the number of samples by splitting each spline into segments with the specified length. The length will be rounded down so that a whole number of samples will fit in each input spline.
  • Length: Evaluate the spline’s points based on the resolution attribute for NURBS and Bézier splines. Has no effect on Poly splines.

Resample Mode: Choose the method for resampling the spline.

  • None: No spline resampling.
  • Count: Sample the specified number of points along each spline.
  • Evaluated: Calculate the number of samples by splitting each spline into segments with the specified length. The length will be rounded down so that a whole number of samples will fit in each input spline.
  • Length: Evaluate the spline’s points based on the resolution attribute for NURBS and Bézier splines. Has no effect on Poly splines.

Resample Count: The number of control points on the new splines.

Resample Length: The approximate length between the control points of the new splines.

Advanced Settings

Advanced Settings: Toggle the visibility of advanced parameters.

  • Disable: Disable the Advanced Settings.
  • Enable: Enable the Advanced Settings.

Split To Instances: Enables the option to split each mesh island as an instance.

Split To Instances Selection: Use custom selection for Split to Instances. You can use an Input Attribute Toggle.

Distance Points Threshold: The distance to use for searching for nearby points.
