Object BFracture

Instance Mode

Choose Instance Mode: Render Instances realize instances, Multi-Instances optimize viewport performance.

  • Instance: The clones are fully-fledged duplicates of the objects to be duplicated. They consume the same amount of memory as if each clone existed as a separate object.
  • Render Instances: Realize all Instances to Real Geometry.
  • Multi-Instances: This mode is particularly suitable for all types of vegetation. When it comes to cloning tens of thousands of trees or tufts of grass, for example, this mode is a must.

Viewport Mode

Some simplified display modes can be defined here for navigation in the Viewport.

  • Off: No instances will be displayed.
  • Points: The instance positions will be displayed with small dots.
  • Matrix: The instance positions will be displayed with small squares (as usual for the matrix object).
  • Bounding Box: The instances will be displayed with their bounding box. This allows the approximate clone sizes to be estimated.
  • Object: The instances will be displayed in full.

Collection Assets

The Collection Assets for BFracture

Mode

BFracture Mode

  • Straight: Each child object will be seen as an Individual Clone.
  • Explode Segments: Each face of the Polygonal Objects will be seen as Clones.
  • Explode Segments & Connect: Each Island of the Polygonal Objects will be seen as Clones.

Separate Children

Output each child of the collection as a separate instance. The list of instances will be sorted alphabetically with the objects and child collections sorted together.

Reset Children

Remove the transform of each of the collection's children when converting them to instances. This is useful in order to keep child objects visually separate in the viewport, while keeping every instance located directly at the location of the point it was added for.

Transformation

Select the Instances from Collection Tranformation

  • Original: Output the instances relative to the collection offset.
  • Relative: Join the input collection instances with the modified object as geometry, maintaining the relative position between the objects in the scene.
