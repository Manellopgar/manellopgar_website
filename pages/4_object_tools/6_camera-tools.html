Object Camera Tool

This Object Tool allows us to delete all instances outside of the camera frame view or use a LOD System with the active Camera.

Fustrum Culling Mode

Camera Tool: Choose the Camera Tool.

  • Fustrum Culling: Deletes the Geometry/Instances Outside the Camera View.
  • LOD: Level Of Detail System based on the Active Camera. Only works with Instances.

Custom Camera: Activates a Custom Camera to use it.

  • Disable: Uses the Active Camera of Scene.
  • Enable: Uses a Custom Camera.

Camera: Drag Your Custom Camera Object.

  • Disable: Uses the Active Camera of Scene.
  • Enable: Uses a Custom Camera.

Only Instances: Affects Only Instances.

Only Geometry: Affects Only Geometry.

Adjust Margin: Adjusts the Margin.

Camera Tool - Fustrum Mode

LOD Mode

LOD Assets: Drag Your Custom Collection with LOD Assets prepared.

LOD Distance: Adjust the LOD Distance between LOD Assets.

Use Only LOD Assets: Enables to Use ONLY the Assets from the Collection LOD.
