BTracer Custom Shape

The BTracer To Mesh tool allows you to define and control the geometry profile of your traced objects with precision before converting them to mesh data.

Set Radius Selection: Custom selection for the Set Radius. You can use Input Attribute Toggle.

Use Spline Factor: Use the Spline Factor as Shape.

Use Custom Shape: If you want a Custom Shape enable this option. Then go to the Geometry Nodes editor and Change the Float Curve in the Red Frame.

Change it in: Change it in Geometry Nodes Editor

Radius: Set the Radius of the Shape.
