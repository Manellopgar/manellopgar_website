BTracer Custom Shape

The BTracer Custom Shape tool allows you to define and control the geometry profile of your traced objects with precision.

Shape Configuration

  • Set Radius Selection: Set a Custom Selection for the Set Radius. You can use Input Attribute Toggle to target specific instances.
  • Use Spline Factor: Use the Spline Factor as Shape, allowing the profile to vary along the length of the curve.
  • Radius: Set the base Radius of the Shape.

Advanced Custom Shapes

For fully custom profiles, you can override the default shape generation:

  • Use Custom Shape: If you want a Custom Shape, enable this option.

Note: When enabled, navigate to the Geometry Nodes editor and modify the Float Curve located in the Red Frame to define your custom profile.
