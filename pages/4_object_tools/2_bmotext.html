Object BMoText

This tool leverages Blender's Native Text system, automatically converting each character into an instance. This ensures full compatibility with all B4D Tools features. Additionally, it allows you to quickly and easily attach text to any Curve.

Quick Tip: If you have a Spline/Curve selected separately from the active object, the addon automatically adds that Spline/Curve to the Spline/Curve parameter.

BMoText - How to Add

Splitting & Configuration

Mode: Choose the Split Letter to Instance Type.

  • Disable: Keeps the Text in its Original Position.
  • Enable: Sets the Text to the Selected Curve Path.

Split to Instances Selection: Splits a selection of Text Letters into groups, then turns each group into an instance. You can use Input Attribute Toggle to control this.

Edge Split Selection: Splits a selection of Text Letters into Face groups, then turns each group into an instance. You can use Input Attribute Toggle to control this.

BMoText - Parameters

Curve Path & Alignment

Curve/Spline Path: Add a Curve Path for the BMoText.

  • Local: The Letters are rotated in Local Space. In other words, they are rotated around the axes described by the initial transform of each Letter.
  • World: The Letters are rotated in World Space. In other words, they are rotated around axes specified in the local space of the modifier object.

Offset: Moves the Text Position along the Curve Path.

Local Offset P.Y: Moves the Text Position in the Curve Path but in the Local Offset P.Y direction.

Rotation Settings

Rotation Coordinate: Choose the Rotation Coordinate system.

  • Local: The Letters are rotated in Local Space. In other words, they are rotated around the axes described by the initial transform of each Letter.
  • World: The Letters are rotated in World Space. In other words, they are rotated around axes specified in the local space of the modifier object.

Rotation: Set the Rotation of the Text on the XYZ Axis.

BMoText - Native Text Parameters

Advanced Options

Custom Selection: Add Custom Selection for the Merge By Distance. You can use Input Attribute Toggle to control this.

Use Merge by Distance: Merges each Letter of Native Text vertices within a given distance, merging surrounding geometry where necessary.

Merge Distance: The distance to use for searching for nearby points.
