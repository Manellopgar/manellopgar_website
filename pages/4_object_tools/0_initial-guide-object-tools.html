Initial Guide for All Fields

In this guide, I explain how to add any Effector and the common parameters that work the same for all Effectors. If you want to know about the specific parameters, please check the other menus for each specific Effector.

By default, all modifiers are in the quick and simple version. If you need the advanced options in all cases, you need to enable the Advanced Settings/Parameters.

? How to add any Field?
▼

Select any mesh object, go to the BCloners Addon panel and click any BCloner button. The selected BCloner and the BCloner Transform modifier is added automatically.

The Fields Modifiers Order

In the Modifier Tab, in almost all cases, the correct order is from top to bottom. Normally, the Addon sets the modifiers in the correct order automatically. However, in some cases, you may need to change this order.
