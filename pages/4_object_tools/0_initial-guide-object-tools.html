Initial Guide for All Object Tools

The Object Tools are different from the rest of tools because they need a specific setup to make it work.

? How to add any Object Tool?
▼

Select any mesh object, go to the BCloners Addon panel and click any Object Tool button. The selected Object Tool is added automatically.

BMoText Tool

This tool doesn't need to have any object selected, because it uses the Blender Native Text and then the addon adds the modifier controls.

BTracer Tool with + Tools

This tool has two + Tools to convert the Spline that is generated by the BTracer into Mesh. If you need the easy version, you can only add the tool BTracer To Mesh, but if you want to have better control with the Shape form that is used by BTracer To Mesh you must use: first the BTracer Custom Shape and then the BTracer To Mesh.
