BTracer To Mesh

The BTracer To Mesh tool allows you to define and control the geometry profile of your traced objects with precision before converting them to mesh data.

Fill Caps: If the profile spline is cyclic, fill the ends of the generated mesh with n-gons.

Material Selection: Set a Custom Selection for the Material. You can use Input Attribute Toggle.

Set Radius Selection: Set a Custom Selection for the Set Radius. You can use Input Attribute Toggle.

Radius: Set the Radius.

Material: Drag Your Custom Material.

Store Proximity: Store Custom Attribute for the Proximity between the Tracer and other Objects.

  • Disable: Doesn't store anything.
  • Enable: Stores the Custom Attribute.

Attribute Name: Define the Attribute Name for the Proximity Mask.

To Min: Adjust the Minimum of the Proximity Mask.

To Max: Adjust the Maximum of the Proximity Mask.
