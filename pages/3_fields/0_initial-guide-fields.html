Initial Guide for All Fields

Here I explain how to add any Field. By default, all Field Modifiers have the first parameter configured in the Enable option. However, if you need to deactivate it, you can switch to Disable; this option deactivates the Field itself.

? How to add any Field?
▼

To add a Field, first ensure you have an active BCloner on your mesh object. Then, go to the BCloners Addon panel and click the desired Field button. The Field modifier will be added automatically to the active BCloner.

The Fields Modifiers order:

In the Modifier Tab, in almost all cases, the correct order is from top to bottom. Normally, the Addon sets the modifiers in the correct order, but in some cases, you need to change it. Here you can see a example to clarify how it works.

How to use the Outliner Collection/Empty automatically created when you add some Fields?

This is a special case for Field Random and Field Sound.

When you add the Field Random, an Empty called "Ctrl_OBJECT_Field" is added automatically. This Empty allows us to control the origin of the Noise Texture when you enable the Random Mode Noise.

Random Field in Noise Mode - Empty Controller

When you add the Field Sound, the Selected Sound is automatically baked into keyframes of the Empty called "Field_Sound_Baked" and this Empty is automatically added to the Sound Baked Object. So by default, everything is working as expected. If you have another Empty or object with the Sound Baked Animation, you can switch it in the parameter Sound Baked Object of the Field Sound.
