Field Set: Vertex Weight

This tool allows you to set and read Vertex Weight from other Blender tools, such as Dynamic Paint. The values are saved as Field to use with B4D Tools.

Field Set Vertex Weight - Example

Vertex Weight Object: Drag the Object with any Vertex Weight or Vertex Group.

Vertex Weight Name: Write the Name used as Vertex Weight or Vertex Group.

Field Set Vertex Weight with Dynamic Paint Brush Type

Min Input: Adjust the Minimum Input Vertex Weight.

Max Input: Adjust the Maximum Input Vertex Weight.

Min Output: Adjust the Minimum Output Vertex Weight.

Max Output: Adjust the Maximum Output Vertex Weight.
