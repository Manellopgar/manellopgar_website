Field: Object Proximity

This tool allows using the proximity of an Object as a Field.

Object/Mesh: Drag the Custom Object.

  • Face Object Field: Calculates the Proximity based on the Object Faces.
  • Edges Object Field: Calculates the Proximity based on the Object Edges.
  • Points Object Field: Calculates the Proximity based on the Object Points.
  • Filled Object Field: Calculates the Proximity based on the Object Volume Filled.

Proximity Mode: Choose the Proximity Mode.

  • Face Object Field: Calculates the Proximity based on the Object Faces.
  • Edges Object Field: Calculates the Proximity based on the Object Edges.
  • Points Object Field: Calculates the Proximity based on the Object Points.
  • Filled Object Field: Calculates the Proximity based on the Object Volume Filled.

Min Distance: Adjust the Minimum Proximity Distance.

Max Distance: Adjust the Maximum Proximity Distance.
