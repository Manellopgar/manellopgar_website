Field: Spline Object

This tool allows using a Spline/Curve Object as a Field.

Field Spline Object Example

Spline/Curve: Drag the Custom Spline/Curve.

Min Distance: Adjust the Minimum Proximity Distance.

Max Distance: Adjust the Maximum Proximity Distance.

Use Spline/Curve Factor: Enable use as Field the Spline Factor Value.

Offset Spline/Curve Field: Add Offset to the Spline Factor Value.
