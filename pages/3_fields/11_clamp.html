Modifier Field Clamp

This modifier field tool allows you to clamp any Field.

Clamp Parameters

Min: Adjust the Minimum value of Effectors Field.

Clamp Min: Clamp to 0 the Minimum value of Effectors Field.

Max: Adjust the Maximum value of Effectors Field.

Clamp Max: Clamp to 1 the Maximum value of Effectors Field.
