Field: Image Tex

This tool allows use the Image Texture as a Field.

Image Tex: Select the Input Image.

Field Controller: Drag your Custom Empty Field Controller.

Invert Image Color: Inverts the Image Input Color.

Use Alpha: Uses the Alpha of the Image as Field.

Invert Image Alpha: Inverts the Image Input Alpha.

Frame: If the Image supports animation, the frame can be set here. This can be keyframed so that the image changes between frames.

Offset: Changes the Position of the Image.

Rotation: Changes the Rotation of the Image.

Scale: Changes the Scale of the Image.

Uniform Scale: Changes the Uniform Scale of the Image.

Use Image Threshold: Enable the control of gradient threshold based on the Image Input.

Image Threshold: Adjusts the Greyscale Threshold based on the Image Input.
