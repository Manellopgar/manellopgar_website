  • Square Square Animation Shape.

Rate

This value defines the time span in frames within which the Time layer will reach a value of 1 or 100%. In other words, smaller values will produce a correspondingly faster value increase (faster effect) and vice-versa.

Offset

This setting can be used to define a temporal offset to the current time. A value of -30, for example, a Time layer effect will initiate at a time of +30.
