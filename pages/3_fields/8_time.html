Field: Time

This tool adds or modifies animation time as a field. You can animate the Field or combine it with other Fields.

Shape: Choose from various shapes that will repeat at a defined point.

  • None: Linear Shape.
  • Sine: Sine Animation Shape.
  • Triangle: Triangle Animation Shape.
  • Sawtooth: Sawtooth Animation Shape.
  • Square: Square Animation Shape.

Time Shapes: Sine - Triangle - Sawtooth - Square

Rate: This value defines the time span in frames within which the Time layer will reach a value of 1 or 100%. In other words, smaller values will produce a correspondingly faster value increase (faster effect) and vice-versa.

Offset: This setting can be used to define a temporal offset to the current time. A value of -30, for example, a Time layer effect will initiate at a time of +30.

  • None: Linear Shape.
  • Sine: Sine Animation Shape.
  • Triangle: Triangle Animation Shape.
  • Sawtooth: Sawtooth Animation Shape.
  • Square: Square Animation Shape.
