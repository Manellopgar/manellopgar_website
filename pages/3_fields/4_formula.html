Field: Formula

This tool allows you to use the Formula Type as a Field.

Formula Type

Formula Type: Choose the Formula Type.

  • Sine: Sine Formula Animation.
  • Wave Texture: Wave Texture Animation.
  • Voronoid Texture: Voronoid Texture Animation.

Wave Type

Wave Type: Choose the Wave Type.

  • Bands X-Y-Z-Diagonal Sine: Wave Texture XYZ-Diagonal Axis Animation.
  • Bands X-Y-Z-Diagonal Saw: Wave Texture XYZ-Diagonal Axis Animation.
  • Bands X-Y-Z-Diagonal Triangle: Wave Texture XYZ-Diagonal Axis Animation.
  • Rings X-Y-Z-Spherical Sine: Wave Texture XYZ-Diagonal Axis Animation.
  • Rings X-Y-Z-Spherical Saw: Wave Texture XYZ-Diagonal Axis Animation.
  • Rings X-Y-Z-Spherical Triangle: Wave Texture XYZ-Diagonal Axis Animation.

Voronoi Type

Voronoi Type: Choose the Voronoi Type.

  • Smooth F1-F1-F2 Euclidean: Smooth F1-F1-F2 Euclidean Texture Animation.
  • Smooth F1-F1-F2 Manhattan: Smooth F1-F1-F2 Manhattan Texture Animation.
  • Smooth F1-F1-F2 Chebychev: Smooth F1-F1-F2 Chebychev Texture Animation.
  • Smooth F1-F1-F2 Minkowski: Smooth F1-F1-F2 Minkowski Texture Animation.
  • Distance to Edge: Distance to Edge Animation.

Sine Axis

Sine Axis: Choose the Axis of Sine Animation.

  • X-Y-Z: Sine X-Y-Z Axis Animation.

Auto Animation

Auto Animation: Auto Animation Options.

  • Disable: Disable Animation Options.
  • Enable: Enable Animation Options.

Animation Controls

Animation Speed: Adjust the Animation Speed.

Loop Animation: Loop Animation options.

  • Disable: Disable Loop Animation Options.
  • Enable: Enable Loop Animation Options.

Frame Loop Out: Enter the End Frame for the Loop Out. If negative, it does not loop.

Amplitude: The amount of the Amplitude Sine.

Offset: Offset the formula.

Scale: Overall texture scale.

Distortion: Amount of distortion of the texture.

Detail: Amount of distortion noise detail.

Detail Scale: Scale of distortion noise.

Detail Roughness: Blend between a smoother noise pattern, and rougher with sharper peaks.

Smoothness (Manhattan): The smoothness of the noise.

Exponent (Minkowski): Exponent of the Minkowski distance metric.

Lacunarity (Voronoi): The difference between the scale of each two consecutive octaves. Larger values correspond to larger scale for higher octaves.

Randomness (Voronoi): The randomness of the noise.

Invert Formula: Inverts the formula values.
