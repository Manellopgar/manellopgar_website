Field: Basic Fields

This Field is added automatically with any Effector and is mandatory for their correct operation. It provides the Base Field upon which you can add as many Effectors as desired. Even with only one Field Basic Field, it drives all Effectors, allowing you to control each Effector's field separately or in combination with others.

Field Type

Choose the Field Type:

  • Infinite Field: Affects all Clones.
  • Linear Field: A linear gradient field whose effect is restricted/masked by its limits in the Square Viewport Gizmos.

Field Linear

  • Object Field: An Object Mesh gradient field whose effect is restricted/masked by its limits in the Objects Viewport Gizmos.

Field Custom Object Shapes

  • Radial Field: A Radial gradient field whose effect is restricted/masked by its limits in the Radial Viewport Gizmos.

Field Radial

Field Radial Softness

  • Factor Field: A linear field whose effect is restricted/masked by Spline Factor from the BCloner.

Field Factor

Field Controller: Drag your Custom Empty Controller.

Object Shape: Drag your Custom Object to use as Field Mesh Shape.
