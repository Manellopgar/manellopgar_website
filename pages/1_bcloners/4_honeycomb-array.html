BCloner HoneyComb Mode

Orientation: Use this setting to define the object coordinate axis to which the honeycomb should be perpendicular. The axis defines the plane on which the honeycomb lies. You can choose between: Z (XY), X (ZY) or Y (ZX).

Offset Direction: Height: Defines the Height offset of every other vertical row.

Offset Direction: Width: Defines the Width offset of every other horizontal row.

Offset (Even): This setting can be used to adjust the offset of every Even other Row.

Offset (Odd): This setting can be used to adjust the offset of every Odd other Row.

Count Width: Defines the amount of Clones Width.

Count Height: Defines the amount of Clones Height.

Mode: Choose between the distance between clones or the dimension of the entire clone conglomerate.

  • Per Step: With this mode, you can determine whether the following parameters Position, Rotation and Scale should apply per clone step. In other words, this mode keeps the distance between clones, is enabled by default.
  • End Point: With this mode, you can determine whether the following parameters Position, Rotation and Scale should apply for the end clone (The values between the start and end clone are then interpolated), in other words, keeps the distance between the first and end clones, adding clones in the middle.

Size Width: Adjusts the Size Width.

Size Height: Adjusts the Size Height.

HoneyComb Array Parameters

Advanced Parameters

Offset Variation (Even): Add Offset Variation in the Offset Animation on the Even Clones.

Offset Variation (Odd): Add Offset Variation in the Offset Animation on the Odd Clones.

Perpendicular Variation: Use this setting to randomly vary the offset of each row in addition to the other honeycomb dimensions (in relation to the Offset Direction value).

Variation Seed: Adjusts the Variation Seed.

Offset Time (Even): Add Offset Time in the Offset Animation on the Even Clones.

Offset Time (Odd): Add Offset Time in the Offset Animation on the Odd Clones.

Offset Time Seed: Adjusts the Offset Time Seed.

Distance Points Threshold: By reducing this parameter, you can delete the clones that are closest to each other.

Use Object Form: Enables the option to use Custom Object Form. This parameter deletes any cloner outside of the volume of that custom object mesh.

Object Form: Drag any Mesh Object to customize the Grid Array Shape.

HoneyComb Array Advanced Parameters

The BCloners Grid Array, Linear, Radial and HoneyComb have the panel called Interface Gizmos. If you open this panel, you see two parameters: Use Size Gizmos and Use Count Gizmos. If you enable these parameters, you MUST have the BCloner Modifier selected to see correctly the Gizmos in the Viewport.

Use Gizmos: Enable to use the Gizmos.

Use Size Gizmos: Enables the Size Gizmos.

Use Count Gizmos: Enables the Count Gizmos.
