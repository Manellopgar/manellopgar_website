BCloner Grid Array Mode

Count X,Y,Z

Number of clones that should be created on the X-axis, Y-axis and Z-axis.

Mode

Choose between clones distance or the dimension of the entire clone conglomerate.

  • •
    Per Step: With this mode, you can determine whether the following parameters Position, Rotation and Scale should apply per clone step. In other words, this mode keeps the distance between clones, is enabled by default.
  • •
    End Point: With this mode, you can determine whether the following parameters Position, Rotation and Scale should apply for the end clone (The values between the start and end clone are then interpolated), in other words, keeps the distance between the first and end clones, adding clones in the middle.
Size XYZ

Use Size to define the extent of the complete grid clone conglomerate in the X, Y and Z directions.

Grid Array Parameters

Advanced Parameters

Fill

By reducing this parameter, You can empty the inside of the shape so that gradually only the clones of the shell remain. The fill option allows deleting the instances randomly.

Fill Seed

Changes the Seed of the Fill parameter.

Distance Points Threshold

By reducing this parameter, You can delete the closest each other clones.

Use Object Form

Enables the option to use Custom Object Form. This parameter deletes any cloner outside of the volume of that custom object mesh.

Object Form

Drag any Mesh Object to customize the Grid Array Shape.

Gizmos

The BCloners Grid Array, Linear, Radial and HoneyComb have the panel called Interface Gizmos. If you open this panel, you see two parameters: Use Size Gizmos and Use Count Gizmos. If you enable these parameters and you MUST have the BCloner Modifier selected to see correctly the Gizmos in the Viewport.
