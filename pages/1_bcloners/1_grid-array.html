BCloner Grid Array Mode

Count X, Y, Z: Number of clones to be created along the X, Y, and Z axes.

Mode

Choose between defining the distance between clones or the total dimension of the clone conglomerate.

  • Per Step: Determines whether Position, Rotation, and Scale parameters apply per clone step. This mode maintains a fixed distance between clones and is enabled by default.
  • End Point: Determines whether Position, Rotation, and Scale parameters apply to the final clone. The values between the start and end clones are interpolated, effectively maintaining the distance between the first and last clones while filling the middle.

Size X, Y, Z: Defines the total extent of the grid clone conglomerate in the X, Y, and Z directions.

Grid Array Parameters

Advanced Parameters

Fill: Reducing this parameter empties the interior of the shape, leaving only the clones of the outer shell. This option allows for random deletion of internal instances.

Fill Seed: Changes the random seed for the Fill parameter.

Distance Points Threshold: Reducing this parameter allows you to delete clones that are too close to each other.

Use Object Form: Enables the use of a Custom Object Form. This parameter deletes any cloner instances that fall outside the volume of the specified custom mesh.

Object Form: Drag any mesh object here to customize the Grid Array shape.

Gizmos

The BCloner Grid Array, Linear, Radial, and HoneyComb tools include a panel called Interface Gizmos. Inside this panel, you will find two parameters: Use Size Gizmos and Use Count Gizmos. To view the gizmos correctly in the viewport, you must have the BCloner Modifier selected.

Use Gizmos: Master toggle for gizmo visibility.

Use Size Gizmos: Enables the gizmos for adjusting size dimensions.

Use Count Gizmos: Enables the gizmos for adjusting clone counts.
