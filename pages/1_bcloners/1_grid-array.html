Count X

Count Y

Number of clones that should be created on the Y-axis

Count Z

Number of clones that should be created on the Z-axis

Mode

Choose between defining clone distance or the dimension of the entire clone conglomerate.

  • Per Step: With this mode, you can determine whether the following parameters (Position, Rotation, and Scale) should apply per clone step. In other words, this mode keeps the distance between clones. It is enabled by default.
  • End Point: With this mode, you can determine whether the following parameters (Position, Rotation, and Scale) should apply for the end clone. The values between the start and end clone are then interpolated, keeping the distance between the first and end clones while adding clones in the middle.

Size XYZ

Use Size to define the extent of the complete grid clone conglomerate in the X, Y, and Z directions.

Visualizing Grid Array parameters

Advanced Parameters

Fill

By reducing this parameter, you can empty the inside of the shape so that gradually only the clones of the shell remain. The Fill option allows deleting instances randomly.

Fill Seed

Changes the Seed of the Fill parameter to generate different random patterns.

Distance Points Threshold

By reducing this parameter, you can delete clones that are closest to each other.

Use Object Form

Enables the option to use a Custom Object Form. This parameter deletes any cloner outside of the volume of that custom object mesh.

Object Form

Drag any Mesh Object here to customize the Grid Array Shape dynamically.

Using multiple geometries for Object Form customization

The BCloners (Grid Array, Linear, Radial, and HoneyComb) feature the Interface Gizmos panel. If you open this panel, you see two parameters: Use Size Gizmos and Use Count Gizmos. To see these gizmos correctly in the Viewport, you MUST have the BCloner Modifier selected.

Use Gizmos

Enables the BCloner Gizmos for direct viewport interaction.

Use Size Gizmos

Enables the Size Gizmos to visually adjust the overall dimensions of the clone array.

Use Count Gizmos

Enables the Count Gizmos to visually modify the number of steps in each axis.
