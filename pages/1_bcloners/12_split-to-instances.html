Split To Instances

This tool splits a selection of geometry elements (such as mesh islands) into groups, then turns each group into an instance. As mentioned before this tool is useful when you have to use Realize Instances Tool and after you want to convert to Instances again.

Mode

Choose the Split Mode

  • Split To Instances: Splits the Instances using the Island Index.
  • Edge Split To Instances: Splits the Instances with Edges Selection. Almost each Face will be an Instance.
Edge Split Selection

Use Custom Edges Selection for Split to Instances. You can use Input Attribute Toggle.

Split To Instances Selection

Use Custom Selection for Split to Instances. You can use Input Attribute Toggle.
