BCloner To Matrix

This tool converts the clones of any BCloner to Matrix Vertex Points. This is useful for example to use any modifier deformer of Blender native in combination with Matrix To Instances tool.

Custom Selection: Set a Custom Selection. You can use Input Attribute Toggle.

BCloner - Attribute Toggle (Used Grid Array as an example)

Matrix To Instances

This tool ONLY Works after the BCloner To Matrix tool or if you have any Mesh to Points in Geometry Nodes modifier ALWAYS place it before. To make easy to remember, this tool can be called also "Matrix to BCloner". As I said before you can use this in combination with any modifier deformer of Blender native.

Asset Type

Choose the Input Asset Type

  • Self: Uses the Self Object as Clone.
  • Object: Uses the Object as Clone.
  • Collection: Uses the Assets of your Collection as Clones.

Custom Selection: Set a Custom Selection. You can use Input Attribute Toggle.

Collection: Drag Your Custom Collection

ID Asset: Offset the index of each Asset.

Randomize Assets: Allows to Randomize the Assets use for the Clones.

Random Seed: Changes the Random Seed

Separate Assets: Active this when you have more than 1 Asset

Reset Coordinates: Reset Position, Rotation and Scale to BEmitter Origin

Use Msh Normal Rotation: Align the Clones Rotation to the Mesh Normal.
