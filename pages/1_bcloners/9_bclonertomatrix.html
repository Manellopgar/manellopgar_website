BCloner To Matrix

This tool converts the clones of any BCloner to Matrix Vertex Points. This is useful for example to use any modifier deformer of Blender native in combination with Matrix To Instances tool.

Custom Selection

Set a Custom Selection. You can use Input Attribute Toggle.

Showing how the BCloner converts to Matrix

Matrix To Instances

This tool ONLY Works after the BClone To Matrix tool or if you have any Mesh to Points in Geometry Nodes modifier ALWAYS place it before. To make easy to remember, this tool can be called also "Matrix to BCloner". As I said before you can use this in combination with any modifier deformer of Blender native.

Showing how the Matrix converts back to Instances

Asset Type

Choose the Input Asset Type

  • Self: Uses the Self Object as Clone.
  • Object: Uses the Object as Clone.
  • Collection: Uses the Assets of your Collection as Clones.
