BCloner Radial Mode

Count

Defines the amount of Clones.

Radius

Radius of Circle generated by Clones.

Axis: XY, ZX or ZY

Sets XY, ZX or ZY as Plane Direction for the Radial Clones.

Align

Align All Cloners to Center.

Start Angle

Start Angle of Circle generated by Clones.

End Angle

End Angle of Circle generated by Clones.

Offset

Offset of Rotation Clones in the Circle.

Radial Parameters

Advanced Parameters

Mix Align

Amount of Align influence.

Spiral Shape

Convert Circle to Spiral Shape.

Rings Mode: Enable

Enables the Rings Mode.

Amount of Rings

Defines the number of Rings.

Rings Points Count

Adjust the Rings Point Count.

Add Rings Points

Adds Extra Points to the Rings.

Distance Points Threshold

By reducing this parameter, you can delete the closest clones to each other.

Use Uniform Rings Distribution

Enables to create a Uniform Distribution of Rings.

Offset Variation

Random Offset of Rotation of Circle. Gives us the possibility of randomising the position of the instances keeping the radial shape. You can change the seed with the Offset Seed parameter.

Offset Seed

Changes the Seed of the Random Offset.

Scale X

Scales in X axis the Radial Shape of Clones.

Scale Y

Scales in Y axis the Radial Shape of Clones.

Radial Advanced Parameters

The BCloners Grid Array, Linear, Radial and HoneyComb have the panel called Interface Gizmos. If you open this panel, you see two parameters: Use Size Gizmos and Use Count Gizmos. If you enable these parameters, you MUST have the BCloner Modifier selected to see correctly the Gizmos in the Viewport.

Use Gizmos

Enable to use the Gizmos.

Use Size Gizmos

Enables the Size Gizmos.

Use Count Gizmos

Enables the Count Gizmos.
