BCloner Object Automatic Detection

Distribution Settings

  • •
    Delete Original
    Deletes the Original Object as Base Mesh.
  • •
    Distribution: Vertex
    A clone on each vertex point.
  • •
    Distribution: Edge
    A clone on each edge (additional settings will be made available).
  • •
    Distribution: Polygon Center
    A clone on each polygon center.
  • •
    Distribution: Surface
    Clones randomly arranged over an object's entire surface.
  • •
    Distribution: Volume
    Randomly dispersed Clones within an object's volume.

BCloner Object Distribution Modes

Alignment & Count

  • •
    Up Vector
    None, +X, -X, +Y, -Y, +Z or -Z. Activating the Align Clone option clearly aligns a clone's Z axis but not the rotation around it. You can choose between these vectors to control orientation.
  • •
    Count (Only on Distribution: Surface or Volume)
    Set the Number of Clones that will be generated.
  • •
    Offset (Only on Distribution: Edge)
    Moves the clones along the edge (values less than 0% or greater than 100% can also be defined, whereby the movement of the clones will not be limited to the edges).
  • •
    Seed (Only on Distribution: Surface or Volume)
    Changes the Seed that changes the position of each Clone of the Distribution.
  • •
    Align Clone
    If this setting is active, each clone will be arranged with its X axis tangential to the Normal Face of the Object. If this setting is not active, each clone will orient itself in accordance with the Cloner's coordinate system.
  • •
    Selection
    Set a Custom Selection. You can use Input Attribute Toggle.

BCloner Object Parameters

Advanced Parameters

  • •
    Use Custom Object
    Enables to Use the Custom Object as Base Mesh.
  • •
    Custom Object
    Drag Your Custom Object to assign it here.
  • •
    Distance Points Threshold
    By reducing this parameter, You can delete the closest each other clones.
  • •
    Use Scale on Edge (Only on Distribution: Edge)
    Enables the Scale on Edge Parameter.
  • •
    Edge Scale (Only on Distribution: Edge)
    The clones will be scaled in the direction of the edge.
  • •
    Enable Scaling (Only on Distribution: Polygon Center or Surface)
    Enables the Scaling parameter.
  • •
    Face Area Scale (Only on Distribution: Polygon Center or Surface)
    Adjust the Face Area Scale of each Clone.
  • •
    Offset Time Seed
    Adjusts the Offset Time Seed for animated distributions.
  • •
    Volume Voxel Size
    Specify the voxel side length for volume distribution.
  • •
    Volume Density
    Number of points to sample per unit volume.
