BCloner Linear Mode

Count: You can define the Number of clones that should be created.

Mode: Choose between the distance between clones or the dimension of the entire clone conglomerate.

  • Per Step: With this mode, you can determine whether the following parameters Position, Rotation and Scale should apply per clone step. In other words, this mode keeps the distance between clones, is enabled by default.
  • End Point: With this mode, you can determine whether the following parameters Position, Rotation and Scale should apply for the end clone (The values between the start and end clone are then interpolated), in other words, keeps the distance between the first and end clones, adding clones in the middle.

Position P.XYZ: Adjust the Position of the clones should be changed for the next clone.

Rotation R.XYZ: Adjust the Rotation of the clones should be changed for the next clone.

Scale S.XYZ: Adjust the Size of the clones should be changed for the next clone.

Linear Parameters

Advanced Parameters

Offset: Shifts the first clone in the direction of subsequent clones, which will shift the entire clone series by the clone positions defined in the Offset value.

Mode: Defines how the rotation change should increase.

  • Single Value: The rotation change should increase evenly from clone to clone.
  • Cumulative Step: Steadily between the first and last clone starting from 0 (Cumulative). In this case, spiral cloning will result.

Step Size: This reduces the step distance between the individual clones, which is tantamount to scaling the entire clone conglomerate (not the clones themselves!) starting from the origin of the BCloner object. You can choose one Step Rotation Axis or combine all.

Step Rotation XYZ: Use the three following rotation input fields to define the curvature of the clone line. Values of 0 will result in no curvature at all, while increasing values will produce a correspondingly greater curvature.

Linear Advanced Parameters

The BCloners Grid Array, Linear, Radial and HoneyComb have the panel called Interface Gizmos. If you open this panel, you see two parameters: Use Size Gizmos and Use Count Gizmos. If you enable these parameters, you MUST have the BCloner Modifier selected to see correctly the Gizmos in the Viewport.

Use Gizmos: Enable to use the Gizmos.

Use Size Gizmos: Enables the Size Gizmos.

Use Count Gizmos: Enables the Count Gizmos.
