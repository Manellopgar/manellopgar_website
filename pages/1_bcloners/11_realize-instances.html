Realize Instances

This tool converts the instances/clones to real mesh. For example, to export as Alembic or other formats. Keep in mind that some other tools have this feature internally applied; in that case, if you need to return to instances, you can use the Split To Instances Tool (check details for this tool in the main menu).

Custom Selection

Set a Custom Selection. You can use Input Attribute Toggle.

Realize All

Realize all levels of nested instances for each top-level instance (overrides the value of the Depth input). You can use Input Attribute Toggle.

Depth

Number of levels of nested instances to realize for each top-level instance.
