BCloner Spline Mode

Note: QUICK TIP: If you have selected the Spline/Curve apart from the active object, the addon automatically adds that Spline/Curve to the Spline/Curve parameter.

BCloner Spline Automatic Detection

Spline Settings

  • Spline/Curve: Drag your Custom Spline/Curve.
  • Delete Spline/Curve: Deletes the Original Spline/Curve.
  • Align: If this setting is active, each clone will be arranged with its X axis tangential to the circle. If this setting is not active, each clone will orient itself in accordance with the Cloner's coordinate system.
  • Distribution Mode: Choose the Distribution Type.
    • Count: Use the neighboring Count setting to define the number of clone segments. Clone spacing will not be uniform, but dependent upon the spline interpolation (i.e. Natural, Even or Adaptive). This result is more often not desired and can be countered by the following modes.
    • Even: Use the neighboring Count setting to define a number of clones. The clones will be arranged with equal intervals along the spline.
    • Step: Use the neighboring Step setting to define the interval between clones along the spline. The spline will be covered in its entirety by clones.
    • Vertex: A Clone will be arranged on each Spline point. No parameters for modification will be available.

BCloner Spline Distribution Modes

  • Count: This setting to define the number of clones to be arranged in the Curve Path.
  • Step: Adjust the Step Interval.
  • Start: Use these settings to limit the Start the arrangement of clones along the spline.
  • End: Use these settings to limit the End the arrangement of clones along the spline.
  • Offset: Use this setting to let the clones move along the circle.
  • Loop: If Offset and Rate are set to values not equal to 0%, clones arranged past the end of the spline will be deleted (Loop not active) or seamlessly (without transition) reinserted at the beginning of the spline (Loop active).

BCloner Spline Parameters

Advanced Parameters

  • Offset Variation: Use this setting to randomize the arrangement of clones for a more coincidental look.
  • Spline Rail: Drag your Custom Spline/Curve as the Spline Rail.
  • Use Rail as Target: Uses the Spline/Curve Rail as the Target Rotation for the Clones.
  • Use Rail as Scale: Uses the Spline/Curve Rail as the Target Scale for the Clones.
  • Rate: Rate basically functions like Offset, with the exception that the clones will move automatically when the animation is played, without having to set keys. This setting defines the distance the clones should travel per second of animation. Activate Smooth Rotation if any irregular movement should occur.
  • Rate Variation: Use this slider to randomly vary the clones’ velocity. If the value defined is high enough, individual clones can even be made to move in reverse. Values less than 0% and greater than 100% can also be defined.
  • Loop with Offset Time: Activates the Loop Mode with Offset Time.
  • Offset Time: Adjust the Offset Time.
  • Random Seed: Changes the Random Seed.

BCloner Spline Advanced Parameters
