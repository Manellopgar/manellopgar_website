Initial Guide for All BCloners

Here I explain how to add any BCloner, which hierarchy outliner is highly recommend to use and also how to use the collection of asset with any BCloner.

? How to add any BCloner?
▼

Select any mesh object, go to the BCloners Addon panel, and click the desired BCloner button. The selected BCloner and the BCloner Transform modifier will be added automatically.

All transformations of each Asset and Object Container:

As well as any other native modifier of Blender, you must apply any transformation PSR (Position, Scale, Rotation) for the correct work of any tool of B4D Tools. In some special cases maybe you don't have to do that, but in 90% of cases it is recommended.

Transformations applied vs Transformations not applied.

The B4D Tools BCloner Modifiers order:

On the Modifier Tab, in almost cases the correct order is from Top to Bottom. Normally the Addon sets the modifiers in the correct order, but in some cases you need to change it. Here you can see a few examples to clarify how it works.

Typical Order example.

Of course you can change manually any modifier (check the left side of the next image) but, You need to keep in mind that almost all tools ONLY work with instances. So the Realize To Instances tools normally will be as the last modifier if you use it alone or as previous modifier of Split To Instances if you use with that tool as well (check the left side of the next image). If you need more info please check the documentation of each tool in the main menu.

Hierarchy order in the Outliner

B4D has a few rules about the hierarchy (sequence and order in the Outliner) that defines exactly which object should be executed. In some cases, the outliner hierarchy in B4D Tools can also be important.

The following could be important factors:

  • Default Hierarchy Structure: Ensure your scene follows the standard structure required by B4D Tools to prevent execution errors.
  • Asset Type: Collection Mode: Pay special attention to object grouping when using this specific mode.

Visual Examples comparison "Correct Hierarchy" vs. "Incorrect Hierarchy".

BCloner Collections

All BCloners have the parameter Collection and by default the addon generates the Drop Your Assets Here Collection and add it to that parameter. The quick and easy way is to add into that collection your custom assets.

Note: But if you need you can change it for any other collection. Also that collection can have other Collections inside but remember B4D Tools and Geometry Nodes only reads the first level, the rest is considered a full object. This is important especially if You use the separate Assets option and the Reset Coordinates option.
