Spline Bed Distribution

This distribution can be loaded directly using the Advanced Distribution button in the BCloners Extension Panel or place it manually if you change the Distribution Menu to Advanced and set the Distribution Object to the parameter Distribution Object. Copies can be arranged along an assigned spline, for example, like pearls on a string.

💡
Quick Tip:

If you have selected an extra Spline/Curve apart from the active object selected, the addon adds automatically the Spline/Curve to the Spline/Curve parameter.

Spline Bed Distribution in action

Distribution Logic

[PARAM] Spline/Curve

Drag Your Custom Spline/Curve.

[PARAM] Clones: Iterate

The distribution automatically runs through all objects in sequence from top to bottom, as they were grouped under the Cloner object. After placing the last object, this sequence starts again with the topmost object under the loner object.

[PARAM] Clones: Random

The object currently to be cloned is selected at random from the objects that have been grouped to the clone object.

[PARAM] Offset

This value is only relevant if several objects have been grouped under the Cloner object or if different shapes are to be cloned. This value determines with which of the objects the sequence of clones along the assigned spline should begin. Please note that the Clones from Distribution option on the Cloner object must be active for this.

Sizing & Counting

[PARAM] Max Count

The number of clones created depends on various factors, such as the length of the assigned spline, the size of the cloned objects and the selected distance to be maintained between the clones. However, this value can be used to specify a maximum number of clones.

Alignment & Rail System

[PARAM] Align

This can be used to influence the rotation of the clones along the spline to align with Tangent Vector.

[PARAM] Rail: Enable

Shows the Rail options.

Spline Bed Rail configuration

[PARAM] Spline Rail

Drag Your Custom Spline/Curve that is used as Spline Rail.

[PARAM] Target

Activating this option, the Z-axes of the clones are aligned with the Rail spline.

[PARAM] Scale To Rail

This can be used to calculate a scaling of the clones in the direction of the assigned Rail spline. Please note that the Align and Target options must also be active for this.

[PARAM] Shift Object

If a Rail spline is assigned and used to scale the clones (Align, Target and Scale To Rail options activated), this can be used to move the position of the cloned objects between the Spline and Rail spline.

Spline Bed Shift Object in action

Gaps & Filling Modes

[PARAM] Gap

This distance is maintained as a gap between the neighboring clones on the spline. As the bounding boxes, i.e., the dimensions of the cloned objects, are taken into account, these gaps are retained even with clones of different sizes.

[PARAM] Fill Gap: Fill (by default)

The clones are always placed along the spline (not entirely) and can only be moved using the following Offset value.

[PARAM] Fill Gap: Full

The clones are always placed along the entire spline and can only be moved using the following Offset value.

[PARAM] Offset

This setting is always available regardless of the selected Fill Gap mode and allows the clones to be moved along the spline.

Ends & Transitions

[PARAM] Start

Adjusts the Start on the Spline where the Clones will be generated.

[PARAM] End

Adjusts the End on the Spline where the Clones will be generated.

[PARAM] Transition Ends

This automatically reduces the size of the copies that come close to the ends of the specified spline section, as demonstrated in the following video.

Scaling & Randomization

[PARAM] Scaling Axis: Direction (by default)

The clones are only scaled in the direction of the spline. This even applies if a Rail is used.

[PARAM] Scaling Axis: Uniform

The clones are scaled evenly in all directions, i.e., proportionally.

Spline Bed Scaling Axis Random in action

[PARAM] Random Scale

This percentage value controls the order of magnitude of the random scaling of the clones. At 0%, the original sizes of the cloned objects are therefore retained.

[PARAM] Seed

This distribution offers random modes based on this value when selecting the objects to be cloned.
