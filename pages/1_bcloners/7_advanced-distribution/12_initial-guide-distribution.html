Initial Guide Advanced Distributions

In this guide, I explain how to add any Advanced Distribution automatically and manually.

? How to add any Advanced Distribution to any BCloner?
▼

Select the mesh object. Ensure it has a BCloner modifier and that the modifier is currently active.

Go to the BCloners Addon panel and click Advanced Distribution. Select your desired distribution from the list. The addon will automatically configure the BCloner to Advanced Distribution mode and generate the object with the selected modifier.

BCloner Adds Any Advanced Distribution

The modifier mainly provides information that can be used to create and place geometry copies or clones. A large part of these functions is also provided by the Advanced Distribution in the BCloners panel of the B4D Addon.

Some of these distribution functions are also provided in the form of capsules that can be combined directly with a MoGraph Cloner Object to control individual distributions of object copies directly via the Outliner. In addition, you can also combine or create your own individual distribution functions directly with the BCloner using the Custom Advanced Distribution option.

BCloner Adds Your Custom Advanced Distribution

All Advanced Distributions have these common parameters which define the Input Assets used in the Advanced Distribution. Keep in mind this works if you have Distribution Enabled, which is enabled by default.

Asset Type

This menu allows us to choose the input Asset Type. You can choose between Object, and Collection. Note that BCloner Object only supports Object and Collection.

  • Object: Uses the added Custom Object from the Object parameter as the clone.
    • Object: Drag your Custom Object that is used as the Clone.
  • Collection: Uses the Objects from the added Collection (Collection parameter) as clones. When this mode is selected, more parameters become available to control the generated clones.
    • Collection: Drag your Custom Collection with the Assets that are used as Clones.
    • Separate Assets: Allows the use of each child of the collection as a separate Clone. Only works if you have more than 1 Asset in the Custom Collection.
    • Reset Coordinates: Reset Position, Rotation, and Scale to BCloner Origin.
