Custom Distribution

This distribution can be loaded directly using the Advanced Distribution button in the BCloners Extension Panel or place it manually if you change the Distribution Menu to Advanced and set the Distribution Object to the parameter Distribution Object.

BCloner Adds Your Custom Advanced Distribution

Parameters

  • Asset Type: Choose the Input Asset Type
    • Object: Uses the Object as Clone.
    • Collection: Uses the Assets of your Collection as Clones.
  • Collection: Drag Your Custom Collection with the Assets that are used as Clones.
  • Object: Drag Your Custom Asset (Mesh Object).
  • Separate Children: Output each child of the collection as a separate instance, sorted alphabetically.
  • Reset Children: Reset the transforms of every child instance in the output. Only used when Separate Children is enabled.
  • Clones: Choose the Clones Mode
    • Iterate: The distribution automatically runs through all objects in sequence from top to bottom, as they were grouped under the Cloner object. After placing the last object, this sequence starts again with the topmost object under the loner object.
    • Random: The object currently to be cloned is selected at random from the objects that have been grouped to the clone object.
  • ID Asset: Offset the index of each Asset.
  • Seed: This distribution offers random modes based on this value when selecting the objects to be cloned.
