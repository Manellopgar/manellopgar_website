Cannonball Distribution

This distribution can be loaded directly using the Advanced Distribution button in the BCloners Extension Panel or place it manually if you change the Distribution Menu to Advanced and set the Distribution Object to the parameter Distribution Object. This distribution enables the stacking of object copies in the form of triangular or square pyramids, for example.

Distribution Modes

  • Clones: Iterate: The distribution automatically runs through all objects in sequence from top to bottom, as they were grouped under the Cloner object. After placing the last object, this sequence starts again with the topmost object.
  • Clones: Random: The object currently to be cloned is selected at random from the objects that have been grouped to the clone object.

Shape Selection

  • Tri Pyramid: A pyramid with a triangular base and three triangles, having a total of four faces.
  • Tri Grid: A grid with a Triangular Shape.
  • Quad Pyramid: A pyramid with a square base and four triangles, having a total of five faces.
  • Quad Grid: A grid with a Quad Shape.

Cannonball Distribution Shapes

Geometry & Dimensions

  • ID Asset: Offset the index of each Asset.
  • Direction: Indicates the direction for the height of the distribution pattern. For example, +Y results in the rectangular base area lying in the XZ plane and the height developing along the positive Y axis.
  • Count X: The number of Clones on the X Axis.
  • Count Y: The number of Clones on the Y Axis.
  • Layers: The Number/Layers of Clones on the Z Axis.

Spacing & Scale

  • Spacing: Adjust the spacing between clones.
  • Scale Source: The distribution evaluates the current size of the cloned objects by calculating their bounding boxes.

Cannonball Spacing

  • Radius: If the Radius Scale Source is selected, this defines the sphere radius used to calculate the distance between clones.
  • Arrangement: Determines how the shape defined under Scale Source is evaluated for the distance calculation.
  • Gap: Reduces or increases the distance between objects evenly in all directions.
  • Seed: Controls random modes based on this value when selecting objects to be cloned.
