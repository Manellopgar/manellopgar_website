BCloner Mirror

This tool is useful if you need to make a mirror of a BCloner, especially it is useful if you use B4D Tools for modeling tasks.

Use Mirror X

Enables the Mirror X Offset.

Offset Mirror X

Values XYZ of Mirror BCloner in the X Axis.

Use Mirror Y

Enables the Mirror Y Offset.

Offset Mirror Y

Values XYZ of Mirror BCloner in the Y Axis.

Use Mirror Z

Enables the Mirror Z Offset.

Offset Mirror Z

Values XYZ of Mirror BCloner in the Z Axis.
