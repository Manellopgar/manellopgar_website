Initial Guide for All BCloners

This guide explains how to add any BCloner, outlines the recommended Outliner hierarchy, and details how to use asset collections effectively with all BCloners.

? How to add any BCloner?
▼

Select any mesh object, go to the BCloners Addon panel, and click any BCloner button. The selected BCloner and the corresponding BCloner Transform modifier are added automatically.

The B4D Tools BCloner Modifiers Order

In the Modifier Tab, the correct order is almost always from top to bottom. Normally, the Addon sets the modifiers in the correct sequence automatically. However, there are cases where you may need to manually adjust this order.

Hierarchy Order in the Outliner

B4D Tools follows specific rules regarding hierarchy (sequence and order in the Outliner) to define exactly which objects should be executed. In some scenarios, the Outliner hierarchy is critical for correct operation.

The following factors are important to consider:

  • Default Hierarchy Structure: Ensure your scene follows the standard structure required by B4D Tools to prevent execution errors.
  • Asset Type: Collection Mode: Pay special attention to object grouping when using this specific mode.
  • Separate Children: The addon must account for separated children geometry correctly; failure to do so can result in missing data or incorrect distributions.

BCloner Collections

All BCloners include a Collection parameter. By default, the addon generates a collection named "Drop Your Assets Here" and automatically assigns it to this parameter.

The quickest and easiest method is to add your custom assets directly into this generated collection.

Advanced Configuration:

  • You can change the parameter to point to any other existing collection if needed.
  • This collection can contain sub-collections inside it.
  • Important Limitation: B4D Tools and Geometry Nodes only read objects/collections at the first level of nesting. Anything deeper is treated as a single, full object.

This limitation is particularly important when using options like Separate Assets or Reset Coordinates.
