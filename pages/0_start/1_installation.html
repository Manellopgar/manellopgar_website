Installation in Blender

Once you have downloaded the addon .zip file from the store where you purchased it:

  • 1. Navigate to Edit > Preferences, then select the Add-ons tab. Click the arrow in the top right corner and select Install from File...

Navigate to Add-ons tab and select Install from File

  • 2. Browse to the add-on .zip file, select it, and click the Install Add-on button.

Browse and select the downloaded .zip file

  • 3. Once installed, click the arrow to expand the add-on preferences.

Expand preferences to verify installation

The add-on is now installed and ready to use. You will find new tabs in the T panel within the Viewport (Toggle the T panel visibility with the T key).

You can always change to the N panel in the Preferences of the addon.

T and N panel locations for B4D Tools

Installation Presets B4D Tools

  • 1. Navigate to Edit > Preferences, then select the File Paths tab, click on the plus button.

Add new file path in Preferences

  • 2. Unzip the Library File, and select the result folder to add as library file.

Select the extracted library folder

  • 3. (OPTIONAL) If you want, rename the File Path to something like B4D Pack Preset.

Optional: Rename the file path

  • 4. Then change any window of layout to Asset Browser.

Switch window layout to Asset Browser

  • 5. Select the Asset Library in Asset Browser or keep active the All Libraries option.

Select the Asset Library

  • 6. Finally, you can see all Presets as Assets in the Asset Browser in the Catalog named Presets Pack.

Presets Pack visible in Asset Browser Catalog

Updating B4D Tools

When updating to a newer version of B4D Tools, simply installing the new version over the old one may leave residual files, potentially causing malfunctions. To prevent this, it's crucial to properly uninstall the previous version first.

  • 1. Uninstall the Previous Version: Open Blender and navigate to Edit > Preferences > Get Extensions. If any version of B4D Tools is installed, uninstall it using the Uninstall button.

Uninstall previous version from Extensions panel
