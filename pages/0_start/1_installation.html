Installation in Blender

Once you have downloaded the add-on .zip file from the store where you purchased it:

  1. Navigate to Edit > Preferences, then select the Add-ons tab. Click the arrow in the top right corner and select Install from File...
  1. Browse to the add-on .zip file, select it, and click the Install Add-on button.
  1. Once installed, click the arrow to expand the add-on preferences.

The add-on is now installed and ready to use. You will find new tabs in the T panel within the Viewport (toggle the T panel visibility with the T key). You can always switch to the N panel via the addon preferences.

Installation Presets B4D Tools

To access the included presets as Assets in Blender:

  1. Navigate to Edit > Preferences, then select the File Paths tab, and click the Add Path (+) button.
  1. Unzip the Library File, select the resulting folder, and add it as a library file path.
  1. (Optional) You can rename the File Path to something like "B4D Pack Preset".
  1. Change any window layout to the Asset Browser.
  1. Select the Asset Library in the Asset Browser or ensure the All Libraries option is active.
  1. Finally, you will see all presets as Assets in the Asset Browser under the catalog named Presets Pack.

Updating B4D Tools

When updating to a newer version of B4D Tools, simply installing the new version over the old one may leave residual files, potentially causing malfunctions. To prevent this, it's crucial to properly uninstall the previous version first.

  1. Uninstall the Previous Version: Open Blender and navigate to Edit > Preferences > Get Extensions. If any version of B4D Tools is installed, uninstall it using the Uninstall button.
  1. Install the New Version: Restart Blender (crucial step) and then install the new add-on file as if it were a fresh installation.

Note: Please refer to the "Installation in Blender" guide for detailed steps on installing the new version.
