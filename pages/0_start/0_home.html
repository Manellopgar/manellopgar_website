Overview

Built with clean, artist-focused controls, it turns complex node setups into simple 1 click actions that work right out of the box.

What is B4D Tools?

B4D Tools is a Blender addon that gives you instant access to mograph tools using the prepared geometry nodes modifiers. Every tool comes with simple and easy sliders and parameters by default; if you enable them, you will have access to the advanced options too.

Frequently Asked Questions

? Do I need to know how to use Geometry Nodes?
▼

Almost not. 90% of the addon can be used without any knowledge of Geometry Nodes. Only a few parameters require you to know how to change easy things in the node tree.

▼

Yes, the BCloners and Full Version. Not for the LITE Version.
