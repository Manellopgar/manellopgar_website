Overview

B4D Tools brings Blender users the motion–design tools they've been waiting for — procedural arrays, fields, falloffs, native text features, animation helpers, and more.

Built with clean, artist-focused controls, it turns complex node setups into simple 1 click actions that work right out of the box.

  • • Supercharges motion-design workflows.
  • • Integrated falloffs and field systems for intuitive animation control.
  • • Ideal for logo animations, product shots, UI motion, and abstract visuals.
  • • Easy for beginners, yet robust enough for seasoned artists.
  • • Time-saving creative presets included!
  • • Discord Server support

What is B4D Tools?

B4D Tools is a Blender addon that gives you instant Access to Mograph Tools using the prepared geometry nodes modifiers. Every tool it has by default the simply and easy sliders and parameters, if you enable you Will have acces to the advanced options too.

Core Features:

BCloners:

6 BCloners with Advanced Distributions

Effectors:

14 Effectors

Fields:

16 Fields (10 Fields + 6 Modifier Fields)

Object Tools:

6 Object Tools + 2 xtra Tools

Material Tools:

6 Material Tools

B4D Tools Requirements:

Requirement Details
Blender Version: 5.0.1 and above
Operation System: Windows, MacOS, Linux
Render Engine: EEVEE and Cycles
