Effector Spline

Spline/Curve

Drag Your Custom Spline/Curve.

Delete Curve/Spline

Deletes the Original Curve/Spline.

Offset

f Strength is set to 100%, modifying Offset will make sure the clones wander along the spline. Lesser Strength values will produce interesting effects.

Start

Use the Start to define Start point of the spline on which the clones should be arranged.

End

Use the Start to define End point of the spline on which the clones should be arranged.

Clamp

If an Offset value not equal to 0% has been selected, this setting determines whether clones that land on a spline's start or end points should stay there, or appear on the opposite end of the spline.

Align

Align all Clones to the Tangent Spline/Curve.

Transform Mode

Choose the Transform Mode

  • Absolute: (Value: 0)
  • Relative to Node: (Value: 0)
  • Relative to Spline: (Value: 0)

Use Position

Use this setting to define whether or not the Effector’s position should be varied

Use Rotation

Use this setting to define whether or not the Effector’s rotation should be varied.

Rotation

Activating Rotation will make the XYZ settings available, with which each clone can be rotated according to its coordinate system.

Spline Rail

Drag Your Custom Spline/Curve to use as Rail.

Use Rail as Target

Use the Rail Spline as the Target for all Clones.

Use Rail as Scale

Use the Rail Spline as the Uniform Scale for all Clones.
