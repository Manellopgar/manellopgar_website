Effector Spline

As the name suggests, this Effector arranges the clones along a Curve/Spline Path.

Effector Spline - Example

Spline/Curve: Drag Your Custom Spline/Curve.

Delete Curve/Spline: Deletes the Original Curve/Spline.

Offset: If Strength is set to 100%, modifying Offset will make the clones wander along the spline. Lesser Strength values will produce interesting effects.

Start: Use the Start parameter to define the Start point of the spline on which the clones should be arranged.

End: Use the End parameter to define the End point of the spline on which the clones should be arranged.

Clamp: If an Offset value not equal to 0% has been selected, this setting determines whether clones that land on a spline's start or end points should stay there, or appear on the opposite end of the spline.

Align: Align all Clones to the Tangent of the Spline/Curve.

Transform Mode

  • Absolute: The clones are organized directly onto the spline, which is exactly what most users want.
  • Relative to Node: Imagine a copy of the spline at each clone (with the spline oriented in the direction of the Z axis), which follows each clone varying the Offset.
  • Relative to Spline: By varying the Offset value, the clones move in unison relative to the spline, without being organized onto it.

Effector Spline - Transform Modes

Use Position: Use this setting to define whether or not the Effector’s position should be varied.

Use Rotation: Use this setting to define whether or not the Effector’s rotation should be varied.

Rotation: Activating Rotation makes the XYZ settings available, allowing each clone to be rotated according to its coordinate system.

Spline Rail: Drag Your Custom Spline/Curve to use as a Rail.

Use Rail as Target: Use the Rail Spline as the Target for all Clones.

Use Rail as Scale: Use the Rail Spline as the Uniform Scale for all Clones.
