Effector Target

As the name suggests, this Effector changes the target of orientation for all Clones.

Effector Target Example

Orientation Settings

Use Pitch: If this setting is activated, a clone's Z axis will always point to the Object Target; if this setting is not active, the rotation's pitch direction will remain unchanged. An up vector can also be set.

Target Mode: Choose the Target Mode.

  • Default Repel: The Repel effect is controlled by the Object Target itself.
  • Proximity Repel: The Repel effect is controlled by the proximity of Object Target.

Object Target: Drag your Custom Object Target.

Reversed Healing: Instead of the positive Z axis, the negative Z axis will attempt to orient itself to the target.

Up Vector: Choose the Up Vector.

  • None: The Up Vector is not defined.
  • Object: The Up Vector is defined by Custom Object.
  • +X,Y,Z Axis: The Up Vector is defined by positive X,Y,Z Axis.
  • -X,Y,Z Axis: The Up Vector is defined by negative X,Y,Z Axis.

Object Up Vector: Drag Your Object Up Vector.

Effector Target - Up Vector

Repel Options

Repel: Show the Repel options.

  • Disable: Disable the Repel Options.
  • Enable: Enable the Repel Options.

Repel Mode: Choose the Repel Mode.

  • All Axis: The Repel Effect affects All Axis.
  • Only X,Y,Z Axis: The Repel Effect only affects the X,Y,Z Axis.
  • Only XY Axis: The Repel Effect only affects the XY Axis.
  • Only XZ Axis: The Repel Effect only affects the XZ Axis.
  • Only YZ Axis: The Repel Effect only affects the YZ Axis.

Repel Type: Choose the Repel Type.

  • Default Repel: The Repel effect is controlled by the Object Target itself.
  • Proximity Repel: The Repel effect is controlled by the proximity of Object Target.

Distance: Defines the minimum distance the clones must have from the target object (or an Effector in other modes) before being displaced.

Distance Strength: Use this slider to adjust the strength of the repulsion.
