Initial Guide for All Effectors

This guide explains how to add any Effector. By default, all modifiers are in the quick and simple version. If you need the advanced options for all cases, you must enable the Advanced Settings/Parameters option.

? How to add any Effector?
▼

Select any mesh object, go to the Effectors Addon panel, and click any Effector button. The selected Effector and the Basic Fields modifier are added automatically.

How to Add Any Effector

The Effectors Modifiers Order

In the Modifier Tab, in almost all cases, the correct order is from top to bottom. Normally, the Addon sets the modifiers in the correct order, but in some cases, you may need to change it for proper functionality.

The most important rule is: You must have 1 Basic Fields Modifier before all Effectors and after any BCloner Transform or BFracture.

Effectors Modifiers Order

Use the Outliner Effector Collections

Effectors: Plain, Random, Step, Image Tex, Noise Tex, Sound, Volume, Formula, and Time have Other Parameters where the Collection Custom_Asset_Modify is automatically assigned to the correct socket. Here you can add any Asset that you want to use as a substitute for the original clones.

Keep in mind that it has the same limitation: Geometry Nodes only reads the first child of that collection. For more information, please check the Initial BCloners Guide.

Special Case Effector Sound

This effector has a two-state button. When you click the button:

  1. It shows a file browser window where you have to select the audio file.
  2. Now click again to return the interface changes to the original state.
