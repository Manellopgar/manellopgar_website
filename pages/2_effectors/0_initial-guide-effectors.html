Initial Guide for All Effectors

Here I explain how to add any Effector and the common parameters that work the same for all Effectors. If you want to know about the specific parameters, please check the other menus for each specific Effector.

By default, all modifiers are in the quick and simple version. If you need the advanced options in all cases, you need to enable the Advanced Settings/Parameters.

? How to add any Effector?
▼

Select any mesh object, go to the BCloners Addon panel and click any BCloner button. The selected BCloner and the BCloner Transform modifier are added automatically.

The Effectors Modifiers Order

In the Modifier Tab, in almost all cases, the correct order is from top to bottom. Normally, the Addon sets the modifiers in the correct order automatically, but in some cases, you may need to change it manually.
