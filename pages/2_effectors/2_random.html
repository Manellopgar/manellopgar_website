Effector Random

As the name suggests, this is an Effector that randomizes clones in various ways.

Effector Random Example

Random Seed: Changes the Seed of the Random.

Invert Random: Inverts the values of the Random.

Random Mode: Choose the Random Mode

  • Random: This mode provides truly Random results.
  • Gaussian: This mode provides somewhat lower values as does Random.
  • Noise: This mode provides truly Random animated results.
  • Turbulence: This mode provides somewhat higher animated values as does Noise.

Effector Random Modes

Noise Scale: Uniform Noise Scale

Auto Animation: Show the Auto Animation settings

Animation Speed: Adjust the Auto Animation Speed.

Use Instance Index: Use the Instance Index to Randomize the Animation.

Loop Animation: Show the Loop Animation settings

Frame Loop Out: Enter the End Frame for the Loop Out. If is negative doesn't loop.

Use Random Time: Enables the Random Time Between Clones.

Rand Time Seed: Changes the Random Animation Time Seed.
