Effector Delay

This Effector introduces time-based delays to clone transformations, creating sequential or spring-like motion effects.

Effector Delay Example

Mode: Choose the Delay Mode.

  • Even: This mode will uniformly blend the clone states. Use the Strength setting to adjust the speed with which this occurs.
  • String: As the name suggests, the clones will behave as though they have springs between them, resulting in slight residual oscillation. Over-oscillation can also occur. Large values will increase residual oscillation, i.e., the larger the value, the softer the springs.

Use Position: Use the Position Animation for the Delay Effect.

Use Rotation: Use the Rotation Animation for the Delay Effect.

Use Scale: Use the Scale Animation for the Delay Effect.

Advanced Settings

Use Frame Range Simulation: Use a Limited Custom Frame Range Simulation.

Start Frame: Set the Start Frame.

End Frame: Set the End Frame.

Delay Settings

Damping Pos: Adjust the damping simulation in the Position parameter.

Stiffness Pos: Adjust the stiffness of the String simulation in the Position parameter.

Damping Rot: Adjust the damping simulation in the Rotation parameter.

Stiffness Rot: Adjust the stiffness of the String simulation in the Rotation parameter.

Damping Scale: Adjust the damping simulation in the Scale parameter.

Stiffness Scale: Adjust the stiffness of the String simulation in the Scale parameter.
