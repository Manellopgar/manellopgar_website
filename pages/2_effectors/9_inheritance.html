Effector Inheritance

As the name suggests, this Effector transfers the PSR (Position, Rotation, Scale) of a source BCloner to another BCloner. This allows you to create morph animations quickly!

Effector Inheritance - Example

Object (Target): Drag Your Custom Object as the Target Object.

Position: Use this setting to define whether or not the Effector’s position should be varied.

Rotation: Use this setting to define whether or not the Effector’s scale should be varied.

Scale: Use this setting to define whether or not the Effector’s rotation should be varied.
