? Is it possible to bake the mesh made with B4D Tools to a static mesh?
▼

Yes, it is possible. You have to do the following:

  • Activate the Render Instances option in BCloner or Last Effector.
  • Select the Object and export the static mesh in any format you like (e.g., .obj).
  • Import the .obj file into Blender or your desired software.
? Is it possible to bake the animation made with B4D Tools?
▼

Yes, it is possible with animations as well. Follow these steps:

  • Activate the Render Instances option.
  • Export animation as an Alembic (.abc) file with the Face Sets option activated for material support.
  • Import the .abc file into Blender.
  • Bake the animation: Go to Object > Animation > Bake Action, then select Visual Keying to Object.
  • Export as an .fbx file.
? Is it possible to combine B4D Tools with MTKA Deformers Toolkit?
▼

Yes, it is possible. You should activate the Render Instances option before using any MTKA Deformers, or add a Geometry Nodes node where you apply Instances.

This step is important to convert all instances into real geometry so that MTKA Deformers can work correctly.
