FAQ

? Which version of Blender is supported?
▼

5.0.1 (Recommended) and above.

? Does B4D Tools work with Instances?
▼

Yes, it works the same as a regular object or asset.

See the example video: B4D Tools with Instances

? Is it possible to bake the mesh made with B4D Tools into a static mesh?
▼

Yes, it is possible. Follow these steps:

  • Activate the Render Instances option in BCloner or the last Effector.
  • Select the object and export the static mesh in your desired format (e.g., .obj).
  • Import the .obj file into Blender or your software of choice.
? Is it possible to bake the animation made with B4D Tools?
▼

Yes, it is possible. Follow these steps:

  • Activate the Render Instances option.
  • Export the animation as an Alembic (.abc) file with the Face Sets option activated for material support.
  • Import the .abc file into Blender.
  • Bake the animation (Object > Animation > Bake Action > Visual Keying to Object).
  • Export as .fbx.
? Is it possible to combine B4D Tools with MTKA Deformers Toolkit?
▼

Yes. You should activate Render Instances before using any MTKA Deformers, or add a Geometry Nodes modifier to convert all instances into real geometry so MTKA Deformers can work correctly.

See a sample video: B4D Tools with MTKA
