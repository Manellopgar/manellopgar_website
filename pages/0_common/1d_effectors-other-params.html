Modify Clone Examples

  • Use Custom Clone: Use a custom collection to replace the original clones.
    • Custom Clone: Drag your custom collection here to use as the source for modifying clones.
  • Offset Time: Creates an animation offset between clones based on time.
  • Offset Velocity: Creates an animation offset between clones based on velocity.
  • Offset Seed: Changes the seed value used for the Offset Time calculation, altering the randomization pattern.
  • Sort Field: Use your custom attribute to sort the order of the clones dynamically.
  • Visibility: Hide clones defined by the Effector based on the active selection or field values.
