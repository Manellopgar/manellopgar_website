Remapping

  • Invert: Inverts the Field.
  • Inner Falloff: Adjusts the transition between the Minimum and Maximum of the Field.
  • Min: Adjusts the Minimum value of the Field.
  • Clamp Min: Clamps the Minimum value to 0.
  • Max: Adjusts the Maximum value of the Field.
  • Clamp Max: Clamps the Maximum value to 1.
  • Soft Falloff: Smooths the Field.
  • Multiplier: Defines the strength of the Remapping exclusively between the Min/Max limits.

Countour Mode

Select the mode for modulating the output values.

  • None: The value transfer occurs linearly.
  • Quadratic: The value transfer is based on a curve defined by the Curve option, controlling the transfer from linear to exponential in both directions.
  • Step: Values are restricted to specific discrete steps. Unlike Quantize, modifying Min and Max changes the value interval while the number of steps remains constant.
  • Quantize: Values are restricted to specific discrete steps. Unlike Step, modifying Min and Max changes the value interval while the number of steps remains constant.
  • Curve: The value transfer is based on a freely definable curve in the Geometry Nodes Editor.
  • Steps: Defines the number of steps for the Step and Quantize modes. For Quantize, this is exact only if Max is set to 100%.
  • Curve: Allows the value transfer to be based on a custom curve defined in the Geometry Nodes Editor.
