Field Remapping Parameters

  • • Invert: Inverts the field values.
  • • Inner Falloff: Adjusts the transition between the Minimum and Maximum values of the field.
  • • Min: Adjusts the Minimum value of the field.
  • • Clamp Min: Clamps the Minimum value to 0.
  • • Max: Adjusts the Maximum value of the field.
  • • Clamp Max: Clamps the Maximum value to 1.
  • • Soft Falloff: Smooths the field transition.
  • • Multiplier: Defines the strength of the remapping exclusively between the Min/Max limits.
  • • Curve: The value transfer takes place based on a freely definable curve in the Geometry Nodes Editor.

Output Modulation Mode

Mode: Select the mode for modulating output values.

  • • None: The value transfer will take place linearly.
  • • Quadratic: The value transfer occurs based on a curve defined by the Curve option, controlling the transition from linear to exponential in both directions.
  • • Step: Values are restricted to certain discrete values. Unlike Quantize, when Min and Max values are modified, the number of steps remains constant while the value interval changes.
  • • Quantize: Values are restricted to certain discrete values. Unlike Step, when Min and Max values are modified, the value interval remains constant while the number of steps changes.
  • • Curve: The value transfer takes place based on a freely definable curve in the Geometry Nodes Editor.

Step & Quantize Settings

Steps: Define the number of steps for the Step and Quantize modes. This represents an exact number of steps for the Step option, and for Quantize only when Max is set to 100%.
