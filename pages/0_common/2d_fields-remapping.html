Remapping

Invert: Inverts the Field.

Inner Falloff: Adjusts the transition between the Minimum and Maximum of the Field.

Min: Adjust the Minimum value of Field.

Clamp Min: Clamp to 0 the Minimum value of Field.

Max: Adjust the Maximum value of Field.

Clamp Max: Clamp to 1 the Maximum value of Field.

Soft Falloff: Smoothest the Field.

Multiplier: Defines the strength of the Remapping exclusively between the Min/Max limits.

Contour Mode

Here you can select the mode with which the output values should be modulated.

  • None: The value transfer will take place linearly.
  • Quadratic: The value transfer will take place based on a curved defined by the Curve option, which controls the transfer from linear to exponential in both directions.
  • Step: Values will be restricted to certain discrete values. The difference to Quantize is visible when the Min and Max values are modified: the number of steps remains constant but the value interval changes.
  • Quantize: Values will be restricted to certain discrete values. The difference to Step can be seen when the Min and Max values are modified: the value interval remains constant but the number of steps changes.
  • Curve: The value transfer takes place based on a freely definable curve in the Geometry Nodes Editor.

Steps: With this setting you can define the steps for the modes Step and Quantize. This is an exact number of steps for the Step option and for Quantize only if Max is set to 100%.

Curve: The value transfer takes place based on a freely definable curve in the Geometry Nodes Editor.
