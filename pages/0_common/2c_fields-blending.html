Layer Blending

Choose the Layer Blending

  • Normal: Overrides the Previous Field.
  • Min: Minimum value between this Field and the Previous Field.
  • Subtract: Subtract this Field with the Previous Field.
  • Multiply: Multiply this Field with the Previous Field.
  • Overlay: Overlay this Field with the Previous Field.
  • Max: Maximum value between this Field and the Previous Field.
  • Add: Add this Field to the Previous Field.
  • Screen: Screen between this Field and the Previous Field.
  • Clip: Clip value between this Field and the Effectors Field.

Render Field Check: This parameter allow us to check visually in Render how the Field is working on each Instance.
