Layer Opacity

Layer Opacity: Adjust the percentage of opacity for the Blending Mode layer.

Layer Blending

Layer Blending: Choose how this field interacts with the previous field or effectors.

  • • Normal: Overrides the previous field.
  • • Min: Returns the minimum value between this field and the previous field.
  • • Subtract: Subtracts this field from the previous field.
  • • Multiply: Multiplies this field with the previous field.
  • • Overlay: Overlays this field with the previous field.
  • • Max: Returns the maximum value between this field and the previous field.
  • • Add: Adds this field to the previous field.
  • • Screen: Applies a screen blending mode between this field and the previous field.
  • • Clip: Clips the value between this field and the effectors field.

Render Field Check

Render Field Check: This parameter allows you to visually inspect how the field is being applied to each instance directly in the Render Viewport.
