BCloner Transform

This modifier is automatically added after any BCloner tool, a feature common to all BCloners. Use this panel to apply global controls across all BCloners or to quickly assign a global material.

Display

Display: Shows Clone Indices in the viewport. This is particularly useful for BMograph Selection, enabling quick and precise selection of specific clones.

Custom Selection

Custom Selection: Set a custom selection criteria. You can add any BMograph Selection, Vertex Paint Field, or attributes like Vertex Paint or Weight Paint using the Input Attribute Toggle.

BCloner - Attribute Toggle (Used Grid Array as an example)

Global Transform

Position X, Y, Z: Move all instances along any axis.

Rotation X, Y, Z: Rotate all instances along any axis.

Scale X, Y, Z & Uniform Scale: Scale all instances along specific axes or uniformly.

Advanced Settings

Advanced Settings: This menu enables advanced options and is disabled by default.

Use as Effector Transform: Activate this if you are using this tool as an Effector Transform with Effector Delay.

Material Override

Use Material: Enable to apply a global custom material to all clones.

Material Selection: Set a custom selection for the material. You can use BMograph Selections, Vertex Paint Fields, or attributes like Vertex/Weight Paint via the Input Attribute Toggle.

Material: Drag and drop your custom material here.
