BCloner Transform

This modifier is automatically added after any BCloner and is common to all BCloners. With this tool, you can change global controls for all clones or add a quick global material.

Display

Displays the clone indices in the viewport. This is extremely useful for B-Mograph Selection to make quick and controllable selections.

Custom Selection

Set a Custom Selection: Add any B-Mograph Selection, Vertex Paint Field, or attributes (e.g., Vertex Paint, Weight Paint). You can use the Input Attribute Toggle for this purpose.

BCloner - Attribute Toggle (Used Grid Array as an example)

Transform Controls

  • Position XYZ: Move all instances along any axis.
  • Rotation XYZ: Rotate all instances around any axis.
  • Scale XYZ & Uniform Scale: Scale all instances along specific axes or uniformly.

Advanced Settings

This menu enables advanced settings (disabled by default).

  • Use as Effector Transform: Activate this if you use this tool as an Effector Transform with Effector Delay.
  • Use Material: Enable to add a global custom material.
  • Material Selection: Set a custom material selection using B-Mograph Selections, Vertex Paint Fields, or attributes like Vertex/Weight Paint via the Input Attribute Toggle.
  • Material: Drag your custom material here to apply it.
