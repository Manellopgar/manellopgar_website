Transform Parameters

  • Use Position: Activates the Position parameters.
    • P. XYZ: Adjusts the XYZ position of clones.
  • Use Rotation: Activates the Rotation parameters.
    • R. XYZ: Adjusts the XYZ rotation of clones.
  • Use Scale: Activates the Scale parameters.
    • S. XYZ: Adjusts the XYZ scale of clones.
    • Uniform Scale: Applies uniform scaling to all clones.
    • Use Absolute Scale: Converts the scale values to absolute numbers.
  • Node/Local Space Transform: Switches between Node Space Transform and Local Space Transform modes.
